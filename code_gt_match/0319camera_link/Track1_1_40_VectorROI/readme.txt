Data format:
[unique id] [camera id] [vector type] [start point] [end point] [3D GPS (midpoint)]

Image annotation
    X-Y = [point type]-[unique id]

unique id
    unique for within vector class (i.e. 0-0 and 1-0 are different)

vector type
    0 = entry (magenta)
    1 = exit  (yellow)
